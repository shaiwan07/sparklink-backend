# Sparklink Dating

A Node.js RESTful API backend for the Sparklink Dating mobile application.

## Features
- User authentication (email, phone, social)
- Profile management
- Matchmaking and discovery
- Questionnaire and preferences
- Video call scheduling
- Club/rewards system
- MySQL database integration
- Swagger (OpenAPI) documentation

## Project Structure
- `/routes` - API route definitions
- `/controllers` - Business logic
- `/models` - Database models
- `/docs` - API documentation (Swagger)
- `/config` - Configuration files

## Setup
1. Copy `.env.example` to `.env` and update with your settings.
2. Install dependencies: `npm install`
3. Start the server: `npm run dev`

## Documentation
- Swagger UI: `/api-docs`
- Postman collection: See `/docs/postman_collection.json`
